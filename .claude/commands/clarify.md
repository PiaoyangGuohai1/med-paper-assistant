
# 潤稿

技能：（無專屬 skill，使用 draft-writing tools）

## 流程

1. `list_drafts()` → 選擇草稿
2. `read_draft(filename)` → 讀取
3. 確認改進類型：

| 類型     | 說明                 |
| -------- | -------------------- |
| grammar  | 文法修正             |
| logic    | 邏輯連貫             |
| tone     | 語氣統一（學術正式） |
| condense | 精簡字數             |
| expand   | 擴充內容             |
| citation | 補充引用             |

4. 執行修改（`patch_draft` / `write_draft`）
5. 🔒 NOVELTY + SELLING POINTS 區塊 → 不可刪改弱化
